# claude-scientific-skills

A Claude Code marketplace for scientific research workflows.

## Install this marketplace

```bash
claude plugin marketplace add siyanpi/claude-scientific-skills
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `literature-review` | Systematic literature review for biomedical and TCM research |

## Install a plugin

```bash
claude plugin install literature-review@claude-scientific-skills
```

## Contributing

To add a new plugin, create a subdirectory under `plugins/` following the Claude Code plugin structure, then update `.claude-plugin/marketplace.json`.
